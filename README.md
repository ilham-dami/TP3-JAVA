# TP3 - Java SE (Programmation Orientée Objet)

## Réalisé par
* Nom : Ilham Dami
* Filière : Ingénierie de Données 1
* Module : Java SE
* Professeur : Mohamed CHERRADI
---

## Objectif du TP

Ce TP a pour objectif de maîtriser les concepts de base de la programmation orientée objet en Java :
* Classes et objets
* Constructeurs
* Encapsulation
* Méthodes
* Attributs (private/public)
---

## Exercice 1 : Classe Point

### Fonctionnalités :
* Création d’un point 
* Affichage du point
* Déplacement du point
* Calcul de la distance 
---

## Exercice 2 : Classe Rectangle

### Fonctionnalités :
* Calcul de la surface
* Calcul du périmètre
---

## Exercice 3 : Classe Etudiant

### Fonctionnalités :
* Affichage des informations (toString)
* Vérification si l’étudiant est admis
---

## Exercice 4 : Nombre Complexe

### Fonctionnalités :
* Addition de deux nombres complexes
* Multiplication
* Affichage
---

## Exercice 5 : Compte Bancaire

### Fonctionnalités :
* Dépôt d’argent
* Retrait
* Transfert
* Affichage du solde
---

## Exercice 6 : Voiture

### Fonctionnalités :
* Accélération (max 240 km/h)
* Freinage
---

## Exercice 7 : Produit

### Fonctionnalités :
* Ajouter un produit
* Modifier un produit
* Supprimer un produit
* Rechercher par ID
* Afficher tous les produits
---

## Bonus : Jeu de devinette

### Fonctionnalités :
* Génération d’un nombre aléatoire
* Interaction avec l’utilisateur
* Messages d’aide (plus grand / plus petit)
* Score selon tentatives
---

## Technologies utilisées
* Java
* Eclipse IDE
